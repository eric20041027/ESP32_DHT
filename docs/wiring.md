# 硬體接線說明

## 元件清單

| 元件 | 型號 | 數量 |
|------|------|------|
| 微控制器 | ESP32 DevKit V1 | 1 |
| 溫濕度感測器 | DHT22 (AM2302) | 1 |
| 上拉電阻 | 10kΩ | 1 |
| 麵包板 | — | 1 |
| 杜邦線 | — | 若干 |

## 接線對照表

| DHT22 腳位 | 連接至 | 說明 |
|-----------|--------|------|
| VCC (Pin 1) | ESP32 3.3V | 低功耗供電 |
| DATA (Pin 2) | ESP32 GPIO4 | 數位訊號線 |
| GND (Pin 4) | ESP32 GND | 接地 |

> **重要**：DATA 腳位與 VCC 之間必須串接 **10kΩ 上拉電阻**，否則訊號不穩定。

## 接線示意圖

```
DHT22                        ESP32 DevKit V1
┌──────────┐                 ┌──────────────────┐
│ VCC  [1] ├─────────────────┤ 3.3V             │
│          │                 │                  │
│ DATA [2] ├────┬────────────┤ GPIO4            │
│          │  10kΩ           │                  │
│          │    └────────────┤ 3.3V (上拉)      │
│ NC   [3] │                 │                  │
│ GND  [4] ├─────────────────┤ GND              │
└──────────┘                 └──────────────────┘
```

## 運作模式切換（GPIO0）

| GPIO0 狀態 | 模式 | 適用場景 |
|-----------|------|---------|
| HIGH（不接線，內建上拉） | 連續上傳模式 | 室內市電，每 30 秒上傳一次 |
| LOW（接 GND） | 深度休眠模式 | 戶外電池，採集後立即休眠 |
